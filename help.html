How To Use This Site


1) When you enter the Make Offer page the system will calculate the cost.

2) The seller will enter their wallet address, if there is sufficient funds you will be provided
with a URL by the seller and be directed to log in via Facebook or Twitter.

3) At the Accept Offer page the buyer can accept or reject the offer.

4) If the buyer accepts the seller will not be permited to change the offer.
The contract is initiated and a confirmation page will appear where the buyer should enter their wallet address.

5) Provided that the seller has sufficient funds,click on BUY and a numerical code would be generated.

6) When the seller enters the code given by the buyer and payments are made, the contract is completed.
